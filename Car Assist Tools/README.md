## Application
使用政府提供的即時更新opendata，包含停車場地址和剩餘車位、加油站地址和油價，直觀的以Google Map呈現在手機上提供汽車駕駛人查詢，再加上額外的行事曆功能，在駕駛經過該路段時提醒備忘事項。
  
  
## Technologies
Android App：Android + Google Map + MySQL的應用
