## Application
為了控制面試時程和安排考生行進的序列，使用即時更新的網頁系統，來監控各考場狀態。包含後台管理匯入資料庫，以及報到系統和考生查詢。

## Technologies
以HTML、JavaScript、jQuery、AJAX、PHP撰寫，資料庫為MySQL，網頁伺服器為Apache。
