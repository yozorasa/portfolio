## Application
以電子方式模仿箱鼓的使用，拍打不同位置得到自訂的鼓聲。使用鐵網分別連上電線，拍打時會通電，透過Arduino控制音效播放。
  
  
## Technologies
Arduino Uno板 + 麵包板，微處理器的Processing軟體設計。
